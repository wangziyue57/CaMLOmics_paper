# CaMLOmics_paper

This repository contains the R code accompanying the manuscript:
> Wang Z, Shao YZ, "Causal machine-learning models for multi-omics mediation reveal survival drivers, therapeutic targets and subgroups in glioma"

## Contents

- `R/`: Core functions for the CaMLOmics framework (multi-omics preprocessing, high-dimensional mediation, latent group-lasso Cox prediction model).
- `analysis/`: Scripts to reproduce the main analyses and figures in the manuscript.
